Master Thesis : Automotive Penetration Testing and Threat Modelling

At Scania in Sweden, I penetration tested Scania vehicles and analysed an open source threat modelling language from an attackers perspective. The penetration testing covered many aspects from CAN Bus injection, Firmware reverse engineering and modification to networking. The threat modelling framework analysed was the Meta-Attack Language (MAL), which produces attack graphs. I validated the attack paths and set about implementing changes to the language to improve detail to make the attack graphs more realistic and useful.

Full thesis paper

Ethical Hacking & Computer Security

In late 2019, I ventured into the world of infosec, taking courses on Ethical Hacking and Computer Security. The Ethical Hacking course at KTH is CTF style with participants given VPN credentials before requiring to work through each machine and corresponding flags to complete the course. Learning was entirely independent and the lectures featured hackers telling their 'war stories'. I enjoyed this course so much and learnt a huge amount, leading to me wanting to pursue a thesis in this area. The second course, Computer Security, taught the fundamentals through coursework featuring firewalls, buffer overflows, cryptography, secure messaging, sql injection, XSS and XSRF, and an exam.

ARM Scalable Vector Extension

While interning at ARM I worked several projects based on the Scalable Vector Extension (SVE). I rewrote some of the zlib compression library, vectorising algorithms with SVE C intrinsics. I also worked on testing and improving the auto-vectorisation capability of the compiler. This work involved C programming, optimisation at the assembly level, benchmarking with multiple micro-architecture simulators and working with Clang.

sve: https://developer.arm.com/docs/dui0965/latest/sve-overview/introducing-sve

Mobile Robotics

This project made me realise just how much manual tuning and heavy use of probabilistic methods are used in consumer robotics currently. This project was the practical side of the CO333 Robotics course. Using a Raspberry Pi, BrickPI interface board and numerous sensors we built our own robots, tuned the motors and sensors then implemented variations of probabilistic motion and sensing using techniques such as Monte Carlo Localisation and SLAM.


code: https://github.com/daniely211/Brickpi
course webpage: https://www.doc.ic.ac.uk/~ajd/Robotics/

Piezo IoT percussion

An IoT project, given a sensor the task was to design a novel IoT device using the MQTT transport protocol. The final product was a device where the user could attach piezoelectric sensors to objects surrounding them, use a web app to assign a sound to sensor and then hit the objects and create their own music with the audio coming from the device using the web app.


code: https://github.com/VasiliosRallis/EmbeddedSystems

HLP Visual2

Contributed to a soon to be Open Source project called Visual2, a User-friendly Educational Assembler and Simulator for ARM UAL. The target language is F#, which is transpiled to Javascript (js) thanks to Fable v2.x. Electron is then used to convert the developed web-app to a cross-platform native application, providing access to platform-level commands (i.e. file-system, path, multiple processes), which are unavailable to (vanilla) browser web-apps. I implemented tools to view the Pipelining of instructions and Branch Prediction (BTB + BPT).

C Compiler

ANSI C89 to MIPS Assembly compiler and C to Python translator. I implemented a subset of the C89 spec including the core functionality {local variables; integer arithmetic; while, if, else conditions; for loops, function calls, multidimensional arrays}. Written in C++, using Flex for the Lexer and Bison for the Parser. This was a really fun solo project to get to understand compilers better.


code: https://github.com/wmvndrscht/EIE2-C-Compiler

CNN Architecture Analysis

A research project under the Circuit and Systems Group at Imperial College Electrical Engineering Department. I investigated relationships in the repetition of calculations between overlapping tiles of an image that are passed through specific CNN architectures (eg VGG-16, Alexnet) to weigh up whether to store previous calculations or recompute at each layer of the CNN. This culminated in a Matlab tool now in use for research group to more efficiently implement CNN architectures on FPGAs.

MIPS Simulator

Given a binary file of MIPS instructions, simulate a MIPS CPU. All written in C++. An interesting project to understand the fundamentals of the MIPS Architecture and instruction encoding.


code: https://github.com/wmvndrscht/EIE2-MIPS-Simulator

FPGA Image Processing

Project to experiment with an FPGA to hardware accelerate an Image Processing Application. We chose to look at the application of drone delivery, detecting a specific pattern (target zone) in any rotation with a tolerance of 30 degrees tilt. Tools : C, Xilinx Zybo Board, Vivado HLS, SDSoC.

Automated CNC

While volunteering in Northern Greece with HelpRefugees a grassroots charity, they received a crowdfunded project called Maslow CNC. This automated cutting complex designs on wood, a time consuming problem for them at the time. A friend and I setup the whole project for them, using the open source designs, we constructed {Designed & Built the wooden chassis - carpentry; Setup the motors; Setup the software for an Arduino}.

A fun project to really test mechanical, electrical and carpentry skills in the Greek heat of 38 degrees celsius!
